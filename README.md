# NYT-Scrape

Project to utilize SQL and connect node JS to it usin routes, controller and ORM. Also utilized handlebars.  Deployed in Heroku as well as git hub.

## Getting Started 

Prerequisites 
node.js installed on your computer.

- express
- body-parse
- path
- mongodb
- mongoose
- axios
- cheerio
- express-handlebars
